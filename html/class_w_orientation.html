Detailed Description

A WOrientation represents a hypothetical object in 3D space. To represent position, the class uses a 3D vector. To represent rotation, the class uses a set of 3 vectors:

  • U (up) vector: Points where the object's "up" direction is
  • L (look) vector: Points to where the object is "looking"
  • R (right) vector: Points to the right of the object The 3 vectors are always perpendicular to one another. Together, they can accurately represent an object's orientation without conflicts.
